# scorpia

Scaffolding for e2e testing with playwright.

## Usage

```sh
npm ci

# In UI/dev mode.
npx playwright test --ui --project todomvc

# Headless.
npx playwright test --project todomvc

# Report viewer, including trace, screenshot, and video.
npx playwright show-report tmp/test/e2e/results
```

## Code Name

Scorpia. One of the twelve colonies in the Battlestar Gallactica universe.
